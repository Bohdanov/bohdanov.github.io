Чем мы занимаемся?

Основным направлением нашей деятельности является создание и сборка бумажных скульптур в технике lowpoly papercraft. Также мы продаем электронные файлы формата pdf (для тех, кто хочет собрать модель из своего материала) и работаем над созданием наборов для самостоятельной сборки (для тех, кто не хочет искать материалы для создания моделей самостоятельно). Комплектация наборов следующая: инструкция по сборке, бумага с нанесенной на неё разверткой, шило для биговки, доска для резки, канцелярский нож с металлической направляющей, металлическая линейка и клей, который разработан на заказ специально для бумажного моделирования.

Насколько долговечны бумажные скульптуры?

Достаточно долговечны для того, чтобы радовать Вас несколько лет. Конечно, нужна не очень высокая влажность помещения, минимум прикосновений и только сухая уборка пыли с модели. Для работы используется бумага плотностью 160 г/м2, а также картон немецкой фирмы Fotokarton плотностью 300 г/м2. Если модель больше двух метров в высоту, то будем использовать гофрокартон либо пластик.
